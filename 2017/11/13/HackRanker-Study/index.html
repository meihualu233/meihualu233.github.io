找到一个刷题的小网站，决定好好学习一下DP。

[Equal]:https://www.hackerrank.com/challenges/equal/problem

题目大意：n堆糖果，每次可以选择一堆不增加（增加的糖果为1、2、5），问最少多少次可以让所有糖果一样多。
解法：除了一堆不增加等价于每次减少一堆的糖果，最终状态为使得所有糖果的数量相同。
问题的关键集中在 如何选择最终的状态。如果只选择让糖果变成所有糖果最少的可能会出现问题。
例如
1 5 5
选择最终状态为1 1 1 需要 4步
选择 0 0 0 为 3步

由经验可以猜测最终状态为｛min,min-1,min-2,min-3,min-4｝中的一个
证明： 假设 最终状态为min-5或者更小,原来的糖果数目为 {min,min+k1,min+k2,…..min+kn}

min-5 到 min 最少需要 1步 ，到 其他不同的堆需要的步数肯定大于等于2
相比较于
min 到 min 最少需要 0步 ，到 其他不同的堆需要的步数可能为1步
所以 min 肯定优于 min-5

矛盾 ，所以只要判断｛min,min-1,min-2,min-3,min-4｝中哪种情况最优即可。

计算方法：
1.可以参考 上楼梯 的DP，即从 min 到 目标点 每次有3中 选择（1、2、5）。
2.可以用贪心 尽量走大步子 ，大步子不行用小步子 ，即 res = (target-start)/5 + (targrt-start)%5/2 +(target-start)%5%2 ;

[Sherlock and Cost]:https://www.hackerrank.com/challenges/sherlock-and-cost/problem

题目大意:一个数列An，构造一个新的数列Bn(满足1<=Bi<=Ai),使得 sum( |b[i+1]-b[i]| )(1<=i<=n)最大。
解法:
1.找最优子结构,设dp[n][0]表示前n个数字的最优情况，且第n个数字选为1，dp[n][1]表示前n个数字在第n个数字选择max的情况下的最优情况。
2.状态转移为 dp[n+1][0] = max( dp[n][0] + 1 - 1 , dp[n][1]+abs(a[n]-1))
dp[n+1][1] = max( dp[n][0]+abs(a[n+1]-1) , dp[n][1]+abs(a[n+1]-a[n]))
3.边界条件 为0 ，结果为 max(dp[n-1][0],dp[n-1][1])

[Candies]:https://www.hackerrank.com/challenges/candies/problem

题目大意：给n个学生的发糖，每个学生有一个分数，分数高的学生得到的糖果要不能币自己周围两个的糖果少。
解法：每个点都受到左右两边的分数影响，但对于每个点而言，都必须满足条件。所以从左往右遍历一边，再从右往左遍历，第一次遍历的结果使得每个点和其右侧的点满足，二次遍历使得左侧满足，取二次得到值中最大即为所求。

[Sam and sub-strings]:https://www.hackerrank.com/challenges/sam-and-substrings/problem

题目大意：给你一个数字字符串求其所有子串的和。
解法：对于123，1出现在个位、十位、百位各一次，2出现在个位、十位各两次；3出现在个位三次。总结一下就是 1111+211+3*1

[Abbreviation]:https://www.hackerrank.com/challenges/abbr/problem

题目大意：两个字符串，字符串1可以删去小写字母，或者将小写变成大写两种操作，判断能否就爱那个字符串一变成二。
解法：从后往前看，串a=”BbbaBB” b=”BABB” ,因为结尾都是大写字母，那么 F(a,b) = F(“BbbaB”,”BAB”)，如果a的结尾是小写，则有两种选择。这就是状态转移方程。

#include

using namespace std;

vector< vector > a,dp;
vector msl,msr,mslt,msrt;

int main()
{
int n,m;
while(cin>>n>>m)
{
msl.resize(m+2,0);
msr = mslt = msrt = msl;

    a.resize(n+2,msl);
    dp = a;

    for(int i=1;i<=n;i++)
        for(int j=1;j<=m;j++)
            cin>>a[i][j];

    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
            msl[j] = max(0,msl[j-1]+a[i][j]);
        for(int j=m;j>=1;j--)
            msr[j] = max(0,msr[j+1]+a[i][j]);

        mslt[1] = dp[i-1][1]+a[i][1];
        dp[i][1] = mslt[1]+msr[2];
        for(int j=2;j<=m;j++)
        {
            mslt[j] = max(mslt[j-1]+a[i][j], dp[i-1][j]+a[i][j]+msl[j-1]);
            dp[i][j] = mslt[j] + msr[j+1];
        }
        msrt[m] = dp[i-1][m]+a[i][m];
        dp[i][m] = max(dp[i][m],msrt[m]+msl[m-1]);
        for(int j=m-1;j>=1;j--)
        {
            msrt[j] = max(msrt[j+1]+a[i][j],dp[i-1][j]+a[i][j]+msr[j+1]);
            dp[i][j]=max(dp[i][j],msrt[j]+msl[j-1]);
        }
    }
    int res = dp[n][1];
    for(int i=2;i<=m;i++)
        res = max(res,dp[n][i]);
    cout<<res<<endl;
}

}
