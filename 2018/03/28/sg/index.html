博弈论 SG函数

  • 算法

以前学过几种博弈，感觉最厉害的就是SG函数了，前几个星期学过这玩意，今天突然发现记得不清楚了，索性整理一下。

必胜点和必败点

一般来说做简单的博弈问题，用一下必败点和必胜点的原理就可以找到规律了。
必败点就是指无论怎么操作肯定gg的状态。
必胜点就是你只要不傻肯定能win的状态。

必败点的下一次操作的状态一定是必胜点n。
必胜点的下一次操作中肯定有一个状态是必败点p。

举个例子：
n个石子，两个人轮流抓，每次要么抓1个要么抓2个，最后抓完的人胜利。
分析先抓的人的情况：
对于n=1、2，这是一个必胜点，先抓的肯定赢
对于n=3，职务是必败点。
1 2 3 4 5 6 7
n n p n n p n
以此类推变能发现规律.

例题：hdu 1847 2147

SG函数

首先定义mex(minimal excludant)运算，这是施加于一个集合的运算，表示最小的不属于这
个集合的非负整数。例如mex{0,1,2,4}=3、mex{2,3,5}=0、mex{}=0。

对于任意状态x,定义SG(x)=mex(S)其中S是x后继状态的SG函数值的集合。
如x有三个后继状态分别为SG(a),SG(b),SG(c)，那么SG(x) = mex{SG(a),SG(b),SG(c)}。

SG(x) =0,当且仅当x为必败点时。

还是上面的例子，我们来求一下不同n的sg函数。
sg[0]=0
n = 1 , 可以取走 1 -{1} 个石子，剩余{0}个 ，sg[1] =mex( sg(0) ) = mex(0) =1
n = 2 , 可以取走 2-{1,2}个石子,剩余{0,1}个, sg[2] =mex(sg(0),sg(1)) = mex(0,1)=2
n = 3 , 可以取走3-{1,2}个石子，剩余(1,2)个，sg[3]=mex(sg(1),sg(2)) =mex(1,2) =0
依此类推
和上面的结果一样。

几个结论：
1.可选步数为1~m的连续整数，直接取模即可，SG(x) = x % (m+1);
2.可选步数为任意步，SG(x) = x；（NIM博弈）
3.可选步数为一系列不一定连续的数，递推处理。
